This is a project that essentially allows the user to upload an image (or not) and paint on it. It is written using HTML5's canvas and JavaScript. Most of the code you see here is starter code provided by Eloquent JavaScript. However, I have used this as a jumping-off point so that I can simply modify the paint environment to add all the functionality my client has requested. Current modifications include auto-closing a free-drawn path, filling the path with a transparent, colored field, and resizing the image the user inputs to the canvas size.
